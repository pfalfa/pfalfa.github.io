ABOUT PFALFA PLATFORM

With decentralized technology, Pfalfa technology has the ability to store and access cloud data with a very high level of security. In addition, with the P2P mechanism, transactions can occur at high speeds.

By providing a range of infrastructure capabilities including decentralized identity service, decentralized graph DB, and IPFS storage gatewway, which can be extended indefinitely. Through fast-synchronizing remote network transmission, Pfalfa allows DApps to perform data management such as computing, transmission, storage, and retrieval in an easy way. In addition, the encryption system applied in this project will protect the privacy of data from users..

OUR SERVICE

Alfa Gold

Like vouchers and electricity tokens that are used as tools to access services (internet and electricity data), Alfa Gold is a blockchain token that is used as a media / platform to access technological features in the Pfalfa platform ecosystem.

Identity Service

Pfalfa uses Protocols and Systems that enable the creation, resolution, and discovery of identities & names that are decentralized throughout the system without a center

Storage Service

Pfalfa provides access where users can enter the application by using a decentralized identity and storing data created in the back-end storage owned by the user

Database Service

Structurally stable data storage, where the database is distributed. There is no single point of error and control because it is connected via the P2P Network. Store structured data namely: metadata, transactions, and blocks using MongoDB-like queries.
